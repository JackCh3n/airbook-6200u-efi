

# airbook-6200u-efi

#### 介绍
支持最新bigsur11.5.1，oc版本是最新的0.7.2，所有驱动全是最新

#### 基本配置

cpu：i5 6200u

内存：镁光单条16g ddr3l

显卡：hd520（核显）

无线网卡：dw1560（bcm94352z）

声卡：alc269vb

显示：lg的显示器(**LQ133T1JW21**) 2560x1440

硬盘：朗科120g固态

系统：bigsur 11.5.1

#### 功能完善度

- [x] 扩展分屏显示
- [x] 睡眠唤醒
- [x] usb定制，usb3达到5g传输速度
- [x] 电池电量显示（显示充电图标，显示电量百分比）
- [x] 内置喇叭外放内放出声，插拔耳机正常
- [x] Wifi 无线连接，蓝牙，airdrop，投屏显示正常
- [x] cpu，硬盘温度监控，功耗监控
- [x] cpu睿频加速
- [x] 亮度调节（f1，f2  自行设置快捷键）
- [x] 开盖唤醒
- [x] 内屏hidpi（自行使用hidpi.sh）
- [x] 摄像头正常使用
- [ ] 内置sd卡识别
- [x] 盒盖睡眠
